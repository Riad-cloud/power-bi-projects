# Portfolio Riad — Page Projets avec sous-pages

## Structure
- `projets.html` : liste des dix projets sous forme de cartes.
- `projets/` : dix sous-pages indépendantes.
- `index.html` : liens de la section « Projets à la une » mis à jour.
- `assets/css/style.css` : styles partagés.
- `assets/js/main.js` : aucun changement nécessaire.

## Images
Les images temporaires se trouvent dans `assets/images/projects/`.
Elles sont affichées sans recadrage avec `object-fit: contain`.

Remplacement conseillé : image 1280 × 800 px, ratio 16:10, PNG ou WebP.
